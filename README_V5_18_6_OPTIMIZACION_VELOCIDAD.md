## Instalación

1. Extrae el ZIP.
2. Ejecuta:

```cmd
ACTUALIZAR_OPTIMIZACION_VELOCIDAD_FUR17_V5_18_6.cmd
```

## Validaciones incluidas en el instalador

- Limpieza de respaldos internos `_backup*`.
- Limpieza de payloads antiguos.
- `npm install`
- `npm run lint`
- `npm run build`

## Nota
