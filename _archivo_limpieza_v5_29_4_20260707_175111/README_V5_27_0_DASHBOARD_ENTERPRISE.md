# SEO LOCAL v5.27.0 — Dashboard Enterprise completo

Este instalador consolida en una sola entrega las fases recomendadas para ampliar el dashboard interno.

## Módulos incluidos
1. Autenticación, usuarios y roles.
2. Dashboard dueño del marketplace.
3. Dashboard agencia con límites por agencia.
4. Gestión modular de perfil de agencia.
5. Servicios FUR-S por agencia.
6. Leads y cotizaciones.
7. Reseñas y moderación.
8. Planes y suscripciones.
9. Categorías y servicios.
10. Métricas, reportes y actividad.
