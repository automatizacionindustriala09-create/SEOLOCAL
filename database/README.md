# Consultas PostgreSQL

Conecta pgAdmin a la base `seo_local` y abre Query Tool.

- `01_inspeccionar_esquema.sql`: tablas, migraciones y conteos.
- `02_consultas_negocio.sql`: consultas útiles del marketplace.

Las escrituras normales deben realizarse mediante la API propia para conservar validaciones y transacciones. No existe ORM ni conexión con Odoo.
