# SEO LOCAL v5.21.4 - NAP e informacion simple en perfil de agencia

Actualizacion visual/funcional de las paginas individuales de agencia.

## Cambios

- El modulo Informacion General de la Agencia se simplifica.
- Se elimina la estructura de subdivisiones internas del bloque informativo.
- Se muestra un texto largo y explicativo generado con los datos reales del perfil de cada agencia.
- Se elimina el modulo lateral Datos de contacto directo.
- Los datos NAP quedan visibles en la barra principal del perfil, junto al logo y nombre de la agencia:
  - Name / Nombre
  - Address / Direccion
  - Phone / Telefono
- Se reutilizan los datos existentes de la BD/API: name, location/city/country y phone.
- No ejecuta migraciones y no modifica PostgreSQL.
- Mantiene servicios conectados a FUR-S, full width y resenas pocket al final.
