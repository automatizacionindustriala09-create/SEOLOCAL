# SEO LOCAL v5.18.10 — Redacción de Contenido Local

Instalador para agregar la página funcional personalizada de **FUR-S-CT-001 — Redacción de Contenido Local (1 página)** al marketplace SEO Local.

## Ruta

```text
http://127.0.0.1:3000/#/servicios/fur-s-ct-001
```

## Instalación

Ejecuta:

```cmd
ACTUALIZAR_CONTENIDO_LOCAL_FUR21_V5_18_10.cmd
```

El instalador crea un respaldo externo en:

```text
%USERPROFILE%\Desktop\SEO_LOCAL_RESPALDOS_EXTERNOS
```

## Validaciones esperadas

```text
npm run lint
npm run build
```
